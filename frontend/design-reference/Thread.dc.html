My sourdough starter is a bioreactor — logging CO₂ with an MH-Z19

T
Tomasz Labiak
@tomek_labs

My sourdough starter lives in a jar on the kitchen counter. Naturally, I put a CO₂ sensor on it. This thread documents the rig, the code, and 48 hours of fermentation data.

Setup

  • MH-Z19B NDIR CO₂ sensor, UART mode
  • Raspberry Pi Zero 2 W
  • BME280 for temperature compensation
  • 3D-printed jar collar to hold the sensor above the starter

The read loop is deliberately boring — poll every ten seconds, average over a minute, log to InfluxDB:

import serial, time

z19 = serial.Serial("/dev/ttyAMA0", 9600)
READ = b"\xff\x01\x86\x00\x00\x00\x00\x00\x79"

while True:
    z19.write(READ)
    hi, lo = z19.read(9)[2:4]
    print(hi * 256 + lo, "ppm")
    time.sleep(10)

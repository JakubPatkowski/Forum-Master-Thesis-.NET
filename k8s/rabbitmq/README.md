# k8s/rabbitmq

RabbitMQ 4 StatefulSet (PSS `restricted`, uid 999) + config + Service. Credentials: `rabbitmq-credentials`
secret, generated by `scripts/deploy.sh` (G14 — never `guest`, it is loopback-only).

**Phase 10c contract:** Service `rabbitmq` carries `labels: {app: rabbitmq}` and named ports
`amqp` 5672 / `management` 15672 / `prometheus` 15692 — point the ServiceMonitor at port name
`prometheus`. NetworkPolicy `30-rabbitmq-allow` already admits the `monitoring` namespace on 15692.
Management UI: `make tunnels` → http://localhost:25672 (never via ingress).
